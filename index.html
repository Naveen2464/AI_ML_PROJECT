The N-Queens Problem places N queens on an N×N board such that no two queens attack each other.

Hill Climbing is a local search algorithm. It begins with a random configuration (one queen per column) and repeatedly moves a queen to a better row in its column to minimize attacks.

Random Restart counters the "local optimum" problem. If no single-queen move reduces attacks, the board resets randomly to try a new path.

Steepest-Ascent Hill Climbing:

function HILL-CLIMBING(problem)
  current = make_node(problem.InitialState)
  loop do
    neighbor = highest_valued_successor(current)
    if neighbor.Value <= current.Value
      return current.State // Local max/min
    current = neighbor
